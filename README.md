# Genifyer & Clairvoyant
データの拡充と意思決定の AI. 2024年6月から.

## 概要
2種類の AI

- データの拡充 "Genifyer"
- 意思決定 "Clairvoyant"

の融合システムによる, 「少ないデータから最良意思決定をする」というテーマに沿ったプロジェクト.  

### Genifyer
Generate + Amplify + er の造語.  
VAE(変分オートエンコーダ)をベースにしたデータ拡充用 AI モデル.

### Clairvoyant
反実仮想機械学習の1種であるオフ方策評価を用いた意思決定促進 AI モデル.
